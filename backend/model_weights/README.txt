This directory holds the trained CNN weights.

FILE: digit_cnn.pth       ← best validation accuracy checkpoint
FILE: digit_cnn_last.pth  ← last epoch checkpoint

Both are generated by:
    python backend/train.py

Run from the PROJECT ROOT, not from inside backend/.

Training time estimates:
  GPU (4 GB VRAM, CUDA 12.1) : ~25–40 minutes
  CPU only                   : ~4–6 hours

After training, start the app:
    python app.py
